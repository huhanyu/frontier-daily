进化 / 自演化（EC 主线）

Learning to Self-Evolve

一句话总结：用 RL 显式训练"自我改进策略"，一个 4B 模型在 Text-to-SQL 和通用问答上超过 GPT-5 和 Claude Sonnet 4.5 的自演化版本。

补充： 提出 LSE 框架，核心是把"自我演化"本身作为可训练目标，而不是依赖更强的 frontier model 做 test-time 优化。4B 参数模型通过学习自演化策略，超越了 GPT-5 驱动的自演化方案。来源： arXiv · 2603.18620

MetaClaw: Just Talk – An Agent That Meta-Learns and Evolves in the Wild

一句话总结：持续元学习框架，让部署后的 agent 通过技能合成 + 策略优化持续进化，不需要本地 GPU，端到端任务完成率提升 8.25 倍。

补充： 两层自适应机制：快（推理时技能注入，从失败中提炼可复用行为）+ 慢（空闲窗口做梯度策略优化）。用 GPT-5.2 和 Kimi-K2.5 实测，Kimi 从 21.4% 提升到 40.6%。来源： arXiv · 2603.17187

ARISE: Agent Reasoning with Intrinsic Skill Evolution

一句话总结：层级 RL 框架，Manager 自动构建技能库、Worker 使用技能执行，技能选择和 RL 目标端到端耦合，不再是"选技能"和"学策略"脱节。

补充： Manager 通过 skill generation rollout 对成功轨迹做结构化摘要，Worker 基于检索到的技能做条件执行。技能库随 RL 训练同步演化。来源： arXiv · 2603.16060

SOAR: Self-Improving Language Models for Evolutionary Program Synthesis

一句话总结：让语言模型通过"搜索-学习"循环持续自我改进，用自身搜索经验训练更好的程序采样策略，不依赖人工训练数据。

补充： 每轮交替：(1) 用当前模型做进化搜索生成和精修程序；(2) 用搜索轨迹微调改进模型。四个关键扩展：通用 Python、迭代精修、更大模型、测试时训练+搜索结合。来源： arXiv · 2507.14172

在线学习 / 持续适应

Online Experiential Learning for Language Models

一句话总结：让语言模型在线从经验中持续学习，不是一次性微调而是动态积累，和自演化主线高度相关。

补充： 关注的是部署后不中断服务的学习范式——模型在执行任务的同时更新自身能力。来源： arXiv · 2603.16856

Trajectory-Informed Memory Generation for Self-Improving Agent Systems

一句话总结：从 agent 执行轨迹中自动生成记忆，解决"agent 做了很多但记不住"的核心问题。

补充： 不是简单把历史日志塞进 context，而是对轨迹做结构化提炼后生成可复用记忆。来源： arXiv · 2603.10600

XSkill: Continual Learning from Experience and Skills in Multimodal Agents

一句话总结：多模态 agent 的持续学习 + 技能积累，把经验沉淀成可复用技能，解决"学了就忘"的问题。

补充： 同时处理视觉和语言输入的技能积累，跨任务复用。来源： arXiv · 2603.12056

Vision-Language-Action Models are Natural Continual Learners with Reinforcement Learning

一句话总结：大预训练 VLA 在持续强化学习里没想象中那么容易灾难遗忘，简单的顺序微调 + LoRA 反而已经很强。

补充： 预训练 + 参数高效适配 + on-policy RL 的组合重塑了稳定性—可塑性权衡，对"自演化机器人"有实用参考价值。来源： arXiv · 2603.11653

LLM / Agent（补充方向）

Multi-Agent Memory from a Computer Architecture Perspective

一句话总结：从计算机体系结构视角审视多智能体记忆系统，提出缓存、分层、一致性等工程化设计方案。

补充： 把多 agent 间的记忆共享类比成分布式系统里的缓存一致性问题，实用性很强。来源： arXiv · 2603.10062

Learning to Reason without External Rewards

一句话总结：不依赖外部奖励信号也能让 LLM 学会推理，通过内在动机驱动推理能力提升。

补充： 如果推理能力可以内生发展而不是靠外部标注驱动，意味着自演化可以在更少人工干预下推进。来源： arXiv · 2505.19590
