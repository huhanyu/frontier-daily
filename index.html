自进化

AgentGA: Evolving Code Solutions in Agent-Seed Space

一句话：在 Agent-Seed 空间中演化代码解决方案——优化任务提示符（Agent Seed），而非直接优化代码。

核心思路：传统代码生成直接优化生成的代码，AgentGA 另辟蹊径：优化"Agent Seed"——即任务提示符 + 可选的父种群档案。核心创新：将代码生成视为演化问题，在 Seed 空间中而非代码空间中搜索。实验验证：在多个代码生成基准上，AgentGA 显著优于基线方法。
启发：从"优化代码"升级到"优化提示符"——这是 Agent 代码生成的新思路。

Towards Self-Evolving Agents via Co-Evolutionary Capability Expansion

一句话：通过共进化实现 Agent 能力自扩展——能力中心框架动态生成工具或 Agent。

核心思路：与经验中心自进化（从交互中学习）不同，能力中心框架关注动态扩展 Agentic 系统的能力边界。核心机制：共进化——Agent 在扩展能力的同时验证新能力的有效性。实验覆盖工具创建、多步推理和长程规划任务。
启发：自进化从"学经验"到"造能力"的升级。

Harness Evolution for Future Prediction Agents

一句话：利用演化方法构建未来预测 Agent——从时间对比中提取内部反馈。

核心思路：未来预测需要对未解决问题的反复推理。论文提出：从时间对比中提取内部反馈——在重复预测同一问题时，比较证据和判断如何随时间变化，提取反馈并更新模型。实验覆盖多个未来预测任务。
启发：预测能力的自进化——让 Agent 从"过去的预测"中学习。

DarwinNet: An Evolutionary Network Architecture for Agent-Driven Protocol Synthesis

一句话：生物启发的自进化网络架构——协议从设计时静态范式向运行时动态演化转变。

核心思路：传统网络协议是设计时静态的，难以适应动态环境。DarwinNet 提出自进化架构：协议在运行时动态演化以适应环境变化。核心创新：借鉴生物进化机制，让协议自主"突变"和"选择"。
启发：网络协议的"自动驾驶"——协议自进化降低人工维护成本。

记忆系统

Experience Compression Spectrum: Unifying Memory, Skills, and Rules in LLM Agents

一句话：统一记忆、技能和规则于单一压缩轴——提出经验压缩谱概念。

核心思路：论文提出 Experience Compression Spectrum，将 Agent 的记忆、技能和规则统一在单一压缩轴上。核心贡献：(1) 形式化定义压缩谱；(2) 将 20+ 系统映射到该谱上，揭示现有系统都未达到最优压缩。关键发现：不同任务需要不同的压缩级别。
启发：记忆/技能/规则不是割裂的——它们是压缩的不同层级。

核心思路：RL 在 LLM 上的核心挑战是信用分配——如何把成功归因到正确的推理步骤。论文系统分析 GRPO、PURE、SPO、SPRO 在数学推理任务上的表现。关键发现：短 CoT、可验证的确定性任务是 RL 方法的"甜点"。

RAG vs 学习

RAG or Learning? Understanding the Limits of LLM Adaptation under Continuous Knowledge Drift

一句话：连续知识漂移下 RAG 与微调的选择——何时用 RAG，何时微调？

核心思路：知识随时间变化（连续知识漂移），模型可能输出过时或不一致的推理。论文研究 RAG vs 微调在连续知识漂移下的表现。关键发现：RAG 在高漂移频率下更优，微调在低频率下更省资源。
启发：没有"一刀切"——漂移频率决定技术选择。
