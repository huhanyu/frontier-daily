Agent 进化

EvoSkills: Self-Evolving Agent Skills via Co-Evolutionary Verification

一句话：通过共进化验证实现 Agent 技能自进化——探索 LLMs 在少样本情况下自主提升专业技能的能力边界。

核心思路：EvoSkills 研究 LLMs 在极其有限标注数据（少样本）下通过自进化提升专业技能。核心机制：共进化验证——Agent 在尝试任务的同时生成验证信号，反过来指导自身改进。附录中系外行星凌星周期检测案例展示了该方法在科学任务中的应用潜力。关键发现：在某些领域，自进化 Agent 可在零外部标注的情况下达到接近全监督的性能。
启发：自进化 + 共进化验证的组合值得关注，特别是对数据稀缺的领域。

AgenticRed: Evolving Agentic Systems for Red-Teaming

一句话：用进化的 Agentic 系统进行红队测试——自主发现并验证 LLM 的安全边界，比静态红队方法覆盖更多攻击向量。

核心思路：红队测试是发现 LLM 安全漏洞的关键手段，但静态红队覆盖范围有限。AgenticRed 引入进化的 Agentic 系统进行自动化红队：多 Agent 分工协作，一部分负责探索新的攻击向量，另一部分负责验证漏洞有效性。关键发现：尽管在同一框架内运作，不同 Agent 配置在漏洞发现率上存在显著差异，说明 Agent 架构设计对红队效果影响很大。进化机制让系统能够自适应地发现新的攻击模式，而非依赖预定义的攻击库。
启发：安全测试本身也在被 Agent 化——进化的红队 Agent 与进化的防御 Agent 之间将形成持续的攻防博弈。

arXiv 2601.13518
RoboPhD: Evolving Diverse Complex Agents Under Tight Evaluation Budgets

一句话：在有限评估预算下进化多样化复杂 Agent——系统比较三种优化范式（Elo 锦标赛选择、GEPA、贪心登山）。

核心思路：LLM 引导的 Agent 进化在 2026 年引发热潮，但评估成本高昂——每个候选需要大量测试。RoboPhD 首次系统比较了三种优化范式：Elo 锦标赛选择（RoboPhD）、Pareto-based 选择（GEPA）和贪心登山。核心发现：在有限评估预算下，如何选择测试用例比选择候选更重要。主动学习选择最具信息量的测试可显著提升进化效率。
启发：评估成本是 Agent 进化的核心瓶颈——主动评估策略对实际部署有直接价值。

记忆系统

MemMachine: A Ground-Truth-Preserving Memory System for LLM Agents

一句话：提出保持事实连续性的持久记忆系统——解决 LLM Agent 在长程交互中遗忘关键事实、丧失个性化的核心问题。

核心思路：LLM Agent 需要持久记忆来维持个性化、事实连续性和长程任务表现。MemMachine 的核心创新在于"事实保真"机制：不是简单存储历史，而是主动识别并固化关键事实，防止在后续推理中被模型自己产生的噪声覆盖。设计包含记忆压缩（保留高价值信息，丢弃冗余）和记忆检索（基于当前任务动态组装记忆上下文）两个关键模块。实验覆盖个性化问答、多轮对话和长程代码生成等场景，验证了记忆系统在事实准确性和任务完成率上的双重提升。
启发："记忆保真"这个角度很值得重视——模型自己生成的内容有时会"覆盖"原始事实，如何区分并保护关键信息是持久记忆设计的关键。

科学发现

AI-Driven Research for Databases

一句话：数据库领域 AI 驱动研究综述——全面调查自进化 AI Agent，连接基础模型与终身高 Agent 系统的新范式。

核心思路：本文是数据库领域首个系统性调查 AI 驱动研究的综述。核心贡献：提出自进化 AI Agent 的统一框架，连接基础模型与终身高 Agent 系统。涵盖四个维度：LLM 驱动的查询优化、自适应索引调整、自动化数据清理和实验自动化。关键发现：当前系统可分为"辅助增强"和"完全自动化"两类，后者仍在早期阶段但潜力巨大。引用了自进化 AI Agent 综述（arXiv:2508.07407）作为理论基础。
启发：数据库作为 AI 研究的重要应用场景，正在经历从"用 AI 优化 DB"到"让 AI 自己优化 DB"的范式转变。

arXiv 2604.06566
Autonomous Agents for Scientific Discovery: Orchestrating Scientists, Language, Code, and Physics

一句话：科学发现自动化 Agent 系统——协调科学家、语言、代码与物理，实现假设发现的自动化。

核心思路：本文探索用自主 Agent 进行科学发现——不仅执行任务，更能生成新假设。系统架构包含四个核心组件：科学家 Agent（领域知识）、语言 Agent（文献理解与生成）、代码 Agent（实验执行）和物理 Agent（仿真验证）。关键创新：知识提取模块从海量文献中抽取隐含假设，形成假设候选池；然后通过跨 Agent 协作验证假设可行性。实验覆盖材料科学和天体物理两个领域，验证了自动化假设发现的潜力。
启发：科学发现的自动化正在从"执行已知流程"向"生成新假设"演进——多 Agent 协作是实现这一目标的关键架构。
