进化 / 自演化（EC 主线）

EvoClaw: Evaluating AI Agents on Continuous Software Evolution

一句话总结：把 AI agent 放进连续软件演化场景后，性能会从一次性任务的高分明显掉到长期维护低分，暴露“会做题但不会持续进化”的核心短板。

补充： 论文用 DeepCommit 从提交历史重建可执行里程碑 DAG，再用 EvoClaw 衡量长期演进中的系统完整性与误差累积。来源： AlphaXiv / arXiv · 2603.13428

CLAG: Adaptive Memory Organization via Agent-Driven Clustering for Small Language Model Agents

一句话总结：让小模型 agent 主动给记忆分簇、写标签、做局部演化，比把所有经验扔进一个大池子更稳，也更像“可持续长脑子”。

补充： CLAG 用 agent 驱动的聚类和两阶段检索降低跨主题干扰，重点解决小模型对无关上下文特别敏感的问题。来源： AlphaXiv / arXiv · 2603.15421

Advancing Multimodal Agent Reasoning with Long-Term Neuro-Symbolic Memory

一句话总结：把神经记忆和符号规则结合，给多模态 agent 一种能长期沉淀、增量更新、还能做演绎推理的“长期记忆体”。

补充： NS-Mem 设计了 episodic / semantic / logic 三层记忆，并用自动知识整理机制持续维护规则层，适合开放世界长期任务。来源： AlphaXiv / arXiv · 2603.15280

SAGE: Multi-Agent Self-Evolution for Generalizable Reasoning

一句话总结：这篇最贴“自演化”字面含义——让 Challenger / Planner / Solver / Critic 共用底座、彼此博弈迭代，形成可迁移的多智能体自进化推理框架。

补充： 本条来自本轮 ultimate-search / grok 搜索结果中的 arXiv 条目，AlphaXiv 本次接口未直接返回该条细节页；因此保留为补充主线项。来源： arXiv HTML · 2603.15255

强化学习（辅助主线）

From Passive Observer to Active Critic: Reinforcement Learning Elicits Process Reasoning for Robotic Manipulation

一句话总结：把视频 MLLM 从“旁观者”训成“过程裁判”，用 outcome-based RL 激发显式过程推理，能更早发现机器人长程操作里的失败苗头。

补充： PRIMO R1 在 RoboFail 上达到 67.0% 准确率，超过文中对比的 OpenAI o1，说明 RL 不只是提最终分数，也能改变中间判断方式。来源： AlphaXiv / arXiv · 2603.15600

Vision-Language-Action Models are Natural Continual Learners with Reinforcement Learning

一句话总结：大预训练 VLA 在持续强化学习里没想象中那么容易灾难遗忘，简单的顺序微调 + LoRA 反而已经很强。

补充： 论文指出预训练、参数高效适配和 on-policy RL 的组合重塑了稳定性—可塑性权衡，这对“自演化机器人”是个很实用的信号。来源： AlphaXiv / arXiv · 2603.11653

MA-VLCM: A Vision Language Critic Model for Value Estimation of Policies in Multi-Agent Team Settings

一句话总结：直接拿预训练视觉语言模型当多智能体 RL 的 centralized critic，可减少从零学价值函数的样本浪费。

补充： 这个方向有意思的点是：把大模型的先验知识注入 critic，而不是硬塞进执行策略，兼顾泛化和部署成本。来源： AlphaXiv / arXiv · 2603.15418

GRPO and Reflection Reward for Mathematical Reasoning in Large Language Models

一句话总结：把“反思”本身变成奖励，能在数学推理里系统提升 LLM 的自检和回看能力，是后训练走向自修正的一条直线路径。

补充： 论文采用四阶段框架，把 GRPO 与 reflection reward、accuracy reward、format reward 结合，强调主动反思而不是被动生成。来源： AlphaXiv / arXiv · 2603.14041

LLM / Agent（辅助）

Spend Less, Reason Better: Budget-Aware Value Tree Search for LLM Agents

一句话总结：不是一味多给 token 和工具调用，而是在推理树里按剩余预算动态调探索 / 利用，低预算下也能做出更像样的 agent 决策。

补充： BAVT 是 training-free 的测试时框架，核心价值在于把“资源意识”嵌进 agent 搜索过程，而不是事后限流。来源： AlphaXiv / arXiv · 2603.12634

Writer-R1: Enhancing Generative Writing in LLMs via Memory-augmented Replay Policy Optimization

一句话总结：在开放式写作这种难打标准答案的任务里，把动态评价标准转成回放式策略优化奖励，可让模型一边写一边更会“复盘”。

补充： MRPO 一手做无额外训练的自反思迭代，一手把自动构造的细粒度标准转为 RL 信号，思路很像可复用的自优化模板。来源： AlphaXiv / arXiv · 2603.15061

Brain-Inspired Graph Multi-Agent Systems for LLM Reasoning

一句话总结：把不同专长的 agent 组织成动态图，再用共享工作区统筹，说明“架构进化”本身就能补足单模型推理上限。

补充： BIGMAS 受 global workspace theory 启发，强调问题自适应图结构和全局编排器，不只靠更长 CoT 硬堆推理。来源： AlphaXiv / arXiv · 2603.15371
