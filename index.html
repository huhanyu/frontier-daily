自进化

Harness Evolution for Future Prediction Agents

一句话：通过时间对比反馈实现未来预测 Agent 的自进化——无需外部奖励，从预测轨迹中提取内部反馈。

核心思路：未来预测需要对未解决难题进行反复推理。论文核心：在重复预测同一问题时，对比证据和判断如何随时间变化，提取反馈信号并更新模型——无需外部奖励信号。关键创新：从"时间对比"中自监督学习预测能力。
启发：自进化不依赖外部反馈，从自身历史中学习。

arXiv 2604.15719
Training LLM Agents for Spontaneous, Reward-Free Self-Evolution

一句话：无奖励信号的自演化 LLM Agent——突破 GRPO 计算瓶颈，在超长'horizon世界知识'生成中实现自进化。

核心思路：GRPO 等在线 RL 算法在超长'horizon 任务上计算代价极高（生成世界知识 K 需要海量推理链）。论文提出自发式自演化方法，无需奖励信号即可驱动模型在长'horizon 任务中持续改进。核心创新：绕过 GRPO 的计算瓶颈，实现真正的自演化训练。
启发：自进化 Agent 的训练不再被 GRPO 的长'horizon 计算代价限制。

arXiv 2604.18131
DarwinNet: An Evolutionary Network Architecture for Agent-Driven Protocol Synthesis

一句话：生物进化启发的网络协议自合成——协议从设计时静态范式转向运行时动态演化。

核心思路：传统网络协议在设计时静态确定，难以适应动态环境。DarwinNet 引入生物进化机制：协议在运行时动态演化（"突变"和"选择"），以适应环境变化。Agent 驱动的协议合成——Agent 自主发现和优化通信协议。
启发：网络协议的"自动驾驶"——降低人工维护成本，协议自管理。

arXiv 2604.01236

多智能体协调

Federation over Text: Insight Sharing for Multi-Agent Reasoning

一句话：联邦学习式多 Agent 推理——各 Agent 在解决不同任务时共享元认知洞察，形成共享知识库。

核心思路：不同 Agent 解决不同任务时，各自产生的元认知洞察（metacognitive insights）可以相互借鉴。论文提出 Federation over Text（FoT）：类似联邦学习，但共享的不是梯度或参数，而是文本形式的元认知洞察。多个 Agent 协作生成共享洞察库，提升整体推理质量。
启发：多 Agent 协作不只共享任务结果，更应共享"思考方式"。

arXiv 2604.16778
MemBuilder: Reinforcing LLMs for Long-Term Memory Construction via Attributed Dense Rewards

一句话：用归因密集奖励构建 LLM 长期记忆——多-Agent 架构分工负责记忆构建与检索。

核心思路：现有 LLM 缺乏持久记忆，MemBuilder 用强化学习训练模型构建长期记忆。架构包含多个 Agent：记忆构建 Agent 负责将交互经验转化为结构化记忆，检索 Agent 负责在需要时召回相关记忆。关键创新：归因密集奖励——每个记忆片段都有明确来源标注，确保记忆可溯源。
启发：长期记忆不是一次性的——需要专门的训练和架构支持。
