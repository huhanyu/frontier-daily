进化 / 自演化方向

RetroAgent: From Solving to Evolving via Retrospective Dual Intrinsic Feedback

一句话总结：用“回溯反思 + 双重内在奖励”让 LLM‑Agent 在长期任务中持续进化，而不是一次性解题。

来源： AlphaXiv · arXiv
🔻 痛点背景：传统 RL 更像一次性“解题”，模型知识沉在参数里，难以持续自我改进。
💡 核心解法：引入回溯自反思机制，生成“数值奖励 + 语言教训”双通道内在反馈，驱动持续探索与显式可检索经验。
🚀 关键启发：把“自演化”作为 Agent 的一等能力，而非事后调参。

How Far Can Unsupervised RLVR Scale LLM Training?

一句话总结：无监督“可验证奖励”能短期提升，但会出现“先升后塌”的极限；外部验证更可扩展。

来源： AlphaXiv · arXiv
🔻 痛点背景：RLVR 试图摆脱标注依赖，但其可扩展性和长期收益未知。
💡 核心解法：系统化分析 URLVR 轨迹，指出内生奖励只是“锐化已有知识”。
🚀 关键启发：外部验证奖励（self‑verification 等）是更稳健的扩展路径。
